## 技术实现

### 图标生成策略
1. **优先从本地缓存加载** - 快速响应
2. **自动生成内置图标** - 使用 PIL 绘制高质量几何图形
3. **保存到本地缓存** - 下次直接加载，无需重新生成

### 内置图标系统
所有图标都是使用 PIL (Pillow) 库动态生成的高质量几何图形，**完全不需要网络连接**：

- **文件夹** → 矩形 + 标签
- **文件** → 矩形 + 折角
- **搜索** → 放大镜（圆形 + 手柄）
- **刷新** → 圆形箭头
- **删除** → 垃圾桶形状
- **设置** → 齿轮（圆形 + 齿）
- **主页** → 房子（三角屋顶 + 矩形主体）
- **确认** → 勾选符号
- **关闭** → X 符号
- **箭头** → 方向箭头（上下左右）
- **加号/减号** → 十字/横线
- **外部链接** → 箭头从方框出去
- **图表** → 柱状图/饼图
- **眼睛** → 眼睛轮廓 + 瞳孔
- **网格/列表** → 网格/横线
- **系统图标** → 电源/锁/盾牌等

所有图标都经过精心设计，在不同尺寸下都能保持清晰和美观。

### 图标尺寸
- 按钮图标：18-20px
- 小图标：16px
- 保持一致的视觉比例

## 使用方法

### 在新窗口中使用图标

```python
from utils.icon_utils import get_icon_manager, Icons

class MyWindow:
    def __init__(self):
        # 初始化图标管理器
        self.icon_manager = get_icon_manager(icon_size=18)

        # 创建带图标的按钮
        button = ctk.CTkButton(
            parent,
            text="搜索",
            image=self.icon_manager.get_icon(Icons.SEARCH),
            compound="left",  # 图标在文字左侧
            command=self.search_action
        )
```

查看 [utils/icon_utils.py](utils/icon_utils.py) 中的 `Icons` 类获取完整列表。
