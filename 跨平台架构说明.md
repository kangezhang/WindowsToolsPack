# 工具箱 v2.0 - 跨平台架构

## ✨ 核心特性

### 1. 智能系统适配
- 自动检测运行系统 (Windows/macOS/Linux)
- 根据系统加载对应功能实现
- Windows → 任务栏托盘
- macOS → 菜单栏应用

### 2. 统一权限管理
**Windows**: UAC 自动提权
**macOS**: osascript 权限请求
→ 一个接口,多平台实现

### 3. 功能模块化
每个工具分离 Windows 和 macOS 实现:
```
WindowsFeature  ←→  MacOSFeature
         ↓               ↓
    Feature.create() 工厂方法
```

## 📦 核心模块

### system_detector.py
- SystemType 枚举
- 系统检测
- 配置路径管理

### permission_manager.py
- Windows: UAC
- macOS: root + osascript

### autostart_manager.py
- Windows: 启动文件夹快捷方式
- macOS: LaunchAgent plist

### tray_manager.py
- Windows: pystray
- macOS: rumps

### feature_base.py
功能基类定义:
- WindowsFeatureBase
- MacOSFeatureBase
- CrossPlatformFeatureBase

## 🛠 添加新工具

### 1. 创建 features/new_tool.py
```python
class WindowsNewTool(WindowsFeatureBase):
    def is_installed(self): pass
    def install(self, exe_path): pass
    def uninstall(self): pass

class MacOSNewTool(MacOSFeatureBase):
    # 同上

class NewTool:
    @staticmethod
    def create():
        if SystemDetector.is_windows():
            return WindowsNewTool()
        elif SystemDetector.is_macos():
            return MacOSNewTool()
```

### 2. 注册到 app/toolbox_app.py
```python
self.features = {
    'copy_path': CopyPathFeature.create(),
    'new_tool': NewTool.create(),
}
```

完成！自动出现在菜单中

## ⚠️ 注意事项

1. macOS 需要辅助功能权限
2. 统一使用 UTF-8 编码
3. 路径使用 os.path 保证兼容
4. 测试两个平台的所有功能
5. macOS 发布需要代码签名
